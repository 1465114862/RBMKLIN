# RBMKLIN
此程序的目的为使用open computer控制hbm核科技（版本X5526）中RBMK反应堆并最大化发热，此反应堆所使用的燃料棒为单个燃料棒，且具有线性中子发射函数。

反应堆较小，可在较前期做出，但发电量较大且波动稳定在约千分之一，使用单个HEP239燃料棒时用工业汽轮机发电量约为8.5MHE/s，使用单个HEAus燃料棒时用工业汽轮机发电量约为19MHE/s，且可随时调整输出。

支持6种燃料。

支持2种蒸汽类型，最方便的普通蒸汽，与燃料利用效率最高的超浓密蒸汽。超浓密的发电较高。

支持自动在储能将满时停机，与储能将空是开机，可以禁用。当支持多种能量储存方式：蒸汽管道自己缓存的超浓密蒸汽，利维坦涡轮电量输出缓存，电池方块，电容。

我正在生存档中使用它，目前还未炸过堆。

## 使用方法
### 1.反应堆搭建
首先，根据你所选燃料，决定你要放置的控制棒数量：

* 1：HEU233,HEP239
* 2：HEAus,HEP241,HEA242
* 3：HES

中子源棒的选择：可以选择最简单的镭中子源棒启动，除了HES需要钋210中子源启动

根据你的需要选择反应堆构型吗，以下构型可根据你的需要整体随意旋转，分为燃料棒有慢化与无慢化，控制棒数量1/2/3，对中子源插入的燃料棒永远无需慢化：

* r:碳化钨中子反射器
* a:硼中子吸收器或者碳化钨中子反射器，你可以自己决定，差别不大
* f:燃料棒(无慢化)
* f':燃料棒(有慢化)
* c:控制棒
* g:石墨慢化剂
* s:蒸汽通道
* i:燃料棒存储棒

无慢化1控制：
```
      r
    a f a
  i   g
      s
r g s f s g r
      s
      c
      g
      r
```
无慢化2控制：
```
      r
    a f a
  i   g
      s
r g s f s c g r
      s
      c
      g
      r
```
无慢化3控制：
```
        r
      a f a
    i   g
        s
r g c s f s c g r
        s
        c
        g
        r
```
慢化1控制：
```
    r
  a f a   i
    s
r s f' s r

```
慢化2控制：
```
    r

```
慢化3控制：
```
      r

```
搭建好反应堆后，摆放下反应堆控制台和起重机控制台，手持反应堆控制台连接装置shift+右键被4个蒸汽通道包围的燃料棒，然后分别右键反应堆控制台和起重机控制台绑定反应堆，并将蒸汽通道模式调成你想要的模式。
### 2.open computer外设链接与搭建
首先你需要一台oc的电脑，显示器、键盘、机箱以及其内的内存，cpu，gpu，硬盘，bios（最好一步到位做T3，省去软盘驱动器，且可插因特网卡下载程序）。

除此之外你只需要1个转运器，与若干线缆。

将转运器放在燃料棒存储棒的正下方，再在其周围放上三个容器，分别用来储存未耗尽燃料棒、耗尽燃料棒、中子源棒，记住这三个容器相转运器的坐标位置关系，后面将用到。

用线缆链接电脑与转运器、反应堆控制台、起重机控制台、储能装置，对于最后一个储能装置，如果你无储能装置或者储能装置就是自带的蒸汽通道，则不用链接。
### 3.根据需求编辑代码的选项
在RBMKLIN.lua中，总共有9行可自定义选项，其中前6行是必填的，你应该检查你自己的反应堆并填入相应信息，后3行则可不修改，根据你的需求设置，其意义与用处已在注释中写明：
```
local neutronSourceRodBox=sides.posx --存放中子源的箱子相对于转置器的位置
local fuelRodBox=sides.posz --存放燃料棒的箱子相对于转置器的位置
local DepFuelRodBox=sides.negx --存放耗尽燃料棒的箱子相对于转置器的位置
local fuelName="HEP239" --燃料类型，可选项："HEAus" "HEP239" "HEU233" "HEP241" "HEA242" "HES"
local steamType="Normal" --蒸汽类型，可选项："UltraDense" "Normal"
local energyStorage="Leviathan" --储能方式，可选项："None" "SteamChannel" "Leviathan" "EnergyStorageBlock" "capacitor"

local fixFluxRatio=1.0 --开机时(设定中子通量/可行的最大中子通量)的初始值，范围(0,1]
local minEnergyRatio=0.1 --给定储能方式最低电量比例，达到此容量时自动开机
local maxEnergyRatio=0.9 --给定储能方式最高电量比例，达到此容量时自动停机
```
### 4.open computer程序写入
填完后，此程序即可运行，接下来的工作是把代码放到游戏中的电脑上，分单机与服务器两种情况：

*i.你玩的是单机，存档在本地:*

打开你游戏中的电脑，用
```
edit 自定义文件名 
```
命令创建一个文件，ctrl+s，ctrl+w保存并退出。

退出并保存，打开```.minecraft\saves\存档名\opencomputers\一串字母和数字，你存档中电脑的id\home```，这时你应该能看到刚刚创建的```自定义文件名```文件，用记事本打开，直接复制黏贴你改好的代码并保存。

重新打开存档，此时```自定义文件名```已编辑好。

*ii.你玩的是联机，存档在服务器:*

此情况需要你的电脑中有因特网卡。

先在你单机的创造模式中，按照单机情况i的步骤编辑好文件。

接着运行
```
pastebin put 自定义文件名 
```
运行完成后其应该输出：
```
pastebin put 自定义文件名
Uploading to pastebin.com... success.
Uploaded as http://pastebin.com/<Pastebin ID>
Run "pastebin get <Pastebin ID>" to download anywhere.
```
然后，进入服务器，在你服务器的电脑上，同样有因特网卡，运行命令
```
pastebin get <Pastebin ID> 随意的文件名
```
即可下载文件，并保存为```随意的文件名```。
### 5.程序运行与操作说明
检查用来储存未耗尽燃料棒的容器是否至少有1个燃料、储存中子源棒的容器是否有至少1个中子源，反应堆供水是否充足。如果有那么可以运行程序。

命令行中直接输入你保存的文件名即可运行程序，开启程序后，默认状态为stop，即停止。

输入r可以使其开始运行，输入s可以使其进行燃料棒冷却与抽出并停止运行，输入x同s，只不过会在停止后自动退出程序。

输入i后可以输入0到1之间数字，输入完后按回车确定，以决定以最大功率的某一比例进行输出，或按i取消输入。


反应堆的启动程序是：先通过起重机把燃料棒运输到最中心，在这个过程中调节控制棒位置，然后用起重机插入中子源点火然后再放回去，控制棒控制中子通量。

关机程序是：控制棒到底让燃料棒中子通量降为接近0，等待燃料棒冷却到可以直接通过合成分解的温度，根据耐久不同，用起重机把已冷却的燃料棒运出到未耗尽或者耗尽燃料棒储存箱。

如果程序遇到错误或者反应堆温度过高/中子通量过大会紧急停机(shutdown)，此时其会迅速降低控制棒并试图用起重机提起燃料棒，如果你没有误操作且我没写bug的话，此状态永远不应出现。如果出现，解决办法：

直接强制关闭电脑，开始手动处理，如果燃料棒柱子里没有燃料棒则燃料棒可能存在起重机内，站在起重机控制台前，上下左右可以控制位置，回车键装载或者放下燃料棒，接下来根据情况你自己处理。


程序中显示的各个数值意义：
* Reactor state: 程序正在进行的工作
* Crane state: 起重机正在进行的工作
* fuel Rod temp: 燃料棒柱体的温度与熔点
* fuel Rod skin temp: 燃料棒的表面温度（推测的实际参与融毁判定的温度）与熔点
* flux in: 入射燃料棒的中子通量
* target flux: 要达到的入射中子通量与其在燃料棒满耐久时对应的值
* target flux/max flux(press i to edit): 可以按i编辑，其为设定的入射中子通量与安全的最大中子通量的比值
* control rod level: 目前控制棒的高度
* enrichment: 燃料棒耐久
* energy storage percentage: 能量储存的百分比
* ticks per cycle: 每运行一次主循环所用的tick，正常情况下大部分时间里此值应该是1或2
* [Run] [Stop] [eXit] 可选指令与目前的状态
